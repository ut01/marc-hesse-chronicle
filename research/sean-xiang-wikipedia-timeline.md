## Technical Achievements and Product Portfolio

### Bloombase StoreSafe
**Product:** Intelligent Storage Firewall  
**Description:** Application-transparent encryption security solution for data-at-rest protection across SAN, NAS, cloud storage, and endpoint systems  
**Key Features:**
- AI-powered threat detection and response (TDR)
- Post-quantum cryptography (PQC) support
- NIST FIPS 140-2 certified cryptographic module
- Support for multiple cipher algorithms: AES, RSA, ECDSA, Camellia, SEED, ARIA, GOST, Kalyna, SM4, and PQC protocols
- IEEE 1619 compliant (storage encryption standard)
- OASIS KMIP compliant (key management)
- PKCS#11 compliant

**Platform Support:**
- Hypervisors: VMware ESXi, Citrix Xen, Oracle VirtualBox, Microsoft Hyper-V, IBM PowerVM, Red Hat KVM
- Cloud platforms: AWS, Microsoft Azure, Google Cloud Platform, Rackspace, VMware vCloud Air, IBM SoftLayer and Bluemix
